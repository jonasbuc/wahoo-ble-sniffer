# UnityIntegration/python — Database helpers & inspection

Small utilities to inspect and make the collector SQLite DB human-friendly.

Files
- `db/pretty_dump_db.py` — prints each table and the first rows, converting
  `recv_ts_ns` to ms and ISO8601 for easy reading. Safe read-only tool.
- `db/create_readable_views.py` — creates SQLite VIEWs named `*_readable`
  (e.g. `headpose_readable`) that expose `recv_ts_ms` and `recv_ts_iso`
  and parsed JSON fields for convenient SQL inspection.
- `populate_test_data.py` — (test helper) inserts synthetic rows into
  `collector_out/vrs.sqlite` (used in tests and local validation).
- `validate_db.py` — runs basic checks (ranges, quaternion norm, JSON parse)
  and prints counts. Useful to sanity-check test data.

Quick start (macOS / zsh)

1) Activate the venv in the repo root:

```bash
. .venv/bin/activate
```

2) Create the readable views (safe; does not change raw tables):

```bash
python UnityIntegration/python/db/create_readable_views.py
```

3) Run a pretty dump that prints human-readable timestamps and table contents:

```bash
python UnityIntegration/python/db/pretty_dump_db.py
```

4) Or query the readable views directly with sqlite3:

```bash
sqlite3 -header -column collector_out/vrs.sqlite "SELECT * FROM headpose_readable LIMIT 10;"
sqlite3 -header -column collector_out/vrs.sqlite "SELECT * FROM sessions_readable LIMIT 10;"
```

Notes & recommendations
- The collector stores timestamps in `recv_ts_ns` (nanoseconds since epoch).
  All helpers convert to `recv_ts_ms` (milliseconds) and `recv_ts_iso` for
  display. Keep the raw nanoseconds in the DB — other code and tests expect it.
- If you want CSV/Parquet exports, load the readable views into pandas and
  use `DataFrame.to_csv()` / `to_parquet()` (requires `pyarrow` for parquet).

Example quick-export (Python):

```python
import sqlite3
import pandas as pd

db = 'collector_out/vrs.sqlite'
con = sqlite3.connect(db)
df = pd.read_sql_query('SELECT * FROM headpose_readable;', con)
df.to_csv('headpose_readable.csv', index=False)
con.close()
```
